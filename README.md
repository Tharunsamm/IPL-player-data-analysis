# IPL-player-data-analysis
Python-based data analysis project exploring IPL player performance trends, salary distribution, and seasonal patterns using NumPy and Matplotlib visualizations.
# IPL Player Data Analysis

This project performs exploratory data analysis on IPL player data to identify trends, patterns, and insights across multiple seasons. The analysis focuses on player performance, salary trends, and seasonal variations using Python-based data analysis and visualization techniques.

## Project Objective
The main objective of this project is to analyze IPL player statistics and visualize patterns in performance and salary distribution over different seasons.

## Tools and Technologies
- Python
- NumPy
- Matplotlib
- Jupyter Notebook

## Key Analysis
- Player salary trend analysis across seasons
- Performance comparison between players
- Visualization of seasonal trends
- Identification of patterns and insights from the dataset

## Project Files
- **IPL Data analysis.ipynb** – Jupyter Notebook containing the complete analysis and visualizations.

## Insights
The analysis helps understand:
- How player performance changes across seasons
- Salary trends and possible correlations with performance
- Visualization of player statistics for better interpretation

## How to Run
1. Clone the repository
2. Open the Jupyter Notebook
3. Run the cells to reproduce the analysis and visualizations
